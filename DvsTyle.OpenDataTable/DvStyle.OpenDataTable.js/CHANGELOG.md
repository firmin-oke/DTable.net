Ce fichier explique comment Visual Studio a créé le projet.

Les étapes suivantes ont été utilisées pour générer ce projet :
- Créez le fichier projet (`DvStyle.OpenDataTable.js.esproj`).
- Créez `launch.json` pour activer le débogage.
- Installez les packages npm : `npm init && npm i --save-dev eslint`.
- Créez `app.js`.
- Mettez à jour le point d’entrée `package.json`.
- Créez `eslint.config.js` pour activer le linting.
- Ajoutez le projet à la solution.
- Écrivez ce fichier.
